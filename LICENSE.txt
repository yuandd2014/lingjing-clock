灵境时钟 用户协议
================

本协议只在保障您的基本权利, 不会诱导您做无关决定。

1. 自由使用
   灵境时钟是开源软件, 您可以自由下载、安装、使用、复制和分发。
   完整源代码遵循开源协议开放, 欢迎在 GitHub 上查看 (项目地址:
   https://github.com/yuandd2014/lingjing-clock)。

2. 不保证质量
   本软件按"现状"提供, 作者不对任何特定用途的适用性作承诺。
   请在重要场景下自行评估风险, 但作者会尽力保持稳定。

3. 数据归属
   本软件所有数据 (天气缓存、设置、位置、Live2D 模型) 存储在您
   本地, 不会上传至任何服务器。
   自动更新仅在您主动启用时检查版本, 您随时可在 设置 中关闭。

4. 反馈与贡献
   欢迎通过 GitHub Issues 提交 bug 报告和功能建议,
   也欢迎通过 Pull Request 贡献代码。
